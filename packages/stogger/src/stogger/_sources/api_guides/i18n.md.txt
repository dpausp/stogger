# Internationalization Module

:::{admonition} Test Coverage: 84.3%
:class: tip

This module has high test coverage and is well-documented.
:::

The `stogger.i18n` module provides internationalization support for stogger, including Austrian, Swiss German, and other dialect translations because logging should be in your native language.

## Quick Start

```python
import stogger
stogger.init_i18n(language="at")

# Translate a key
from stogger import t
message = t("success")  # Returns Austrian translation

# With variables
message = t("file_not_found", "errors", filename="test.log")
```

## NicestlogTranslator

The main translator class with dialect support and fallback handling.

```python
from stogger.i18n import NicestlogTranslator

translator = NicestlogTranslator(language="at")
translated = translator.get("success")
```

### Translation Lookup

The translator uses a three-step lookup:

1. Try the requested language
2. Fall back to English
3. Return the key itself as last resort

```python
# Returns Austrian translation if available
translator.get("welcome")

# Falls back to English
translator.get("some_english_only_key")

# Returns "section.key" if not found anywhere
translator.get("completely_unknown_key")
```

### Changing Language

```python
translator.set_language("ch")  # Switch to Swiss German
```

## Convenience Functions

### init_i18n

Initialize the global translator instance.

```python
import stogger
stogger.init_i18n(language="at")
```

### get_translator

Get the current translator instance (auto-initializes if needed).

```python
from stogger import get_translator
translator = get_translator()
```

### t

Shorthand for translation.

```python
from stogger import t

# Simple key lookup
t("success")

# With section
t("file_not_found", "errors")

# With variable substitution
t("user_greeting", name="Alice", count=42)
```

## Translation Files

Translations are loaded from TOML files in the translations directory:

```
translations/
  en.toml      # English (fallback)
  at.toml      # Austrian German
  ch.toml      # Swiss German
  wollgesisch.toml  # Bavarian dialect
```

### TOML Format

```toml
# translations/en.toml
success = "Success!"
user_greeting = "Hello, {name}! You have {count} messages."

[errors]
file_not_found = "File {filename} not found!"
permission_denied = "Permission denied: {path}"
```

### Configuration

Set the translation directory in `pyproject.toml`:

```toml
[tool.stogger]
language = "at"
```

Or configure programmatically:

```python
from stogger.config import StoggerConfig
config = StoggerConfig(
    translation_dir=Path("./translations"),
    language="at"
)
```

## Austrian Shortcuts

Fun convenience functions for adding Austrian flair:

```python
from stogger import oida, leiwand, arsch

oida("something went wrong")    # "Oida, something went wrong!"
leiwand("this code")            # "this code - leiwand!"
arsch("this bug")               # "this bug - des is arsch!"
```
